# Tino Musikavanhu - Portfolio Website

A modern, responsive portfolio website showcasing the work of Tino Musikavanhu - Quantum Architect, Systems Engineer, and Ethical Technologist.

## 🚀 Features

- **Modern Design**: Built with Next.js 14, React, and Tailwind CSS
- **Responsive**: Fully responsive design that works on all devices
- **Interactive**: Smooth animations with Framer Motion
- **Performance**: Optimized for speed and SEO
- **Accessible**: Built with accessibility best practices

## 🛠️ Tech Stack

- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: Ready for Vercel, Netlify, or any static hosting

## 🏃‍♂️ Quick Start

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Open in Browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## 🎨 Customization

### Colors
The website uses a custom color palette defined in `tailwind.config.js`. Key colors:
- **Primary**: Blue gradient (quantum theme)
- **Secondary**: Purple accents
- **Background**: Dark slate tones
- **Text**: Light slate for readability

### Sections
The portfolio includes the following sections:
- **Hero**: Introduction and call-to-action
- **About**: Professional summary and key areas
- **Experience**: Work history and achievements
- **Research**: Publications and breakthrough research
- **Skills**: Technical expertise organized by category
- **Projects**: Featured projects and systems
- **Contact**: Contact information and links

### Animations
Smooth scroll animations are implemented using Framer Motion:
- Fade-in effects for sections
- Hover animations for interactive elements
- Progress bar indicating scroll position
- Quantum-themed particle animations

## 🚀 Deployment

### Vercel (Recommended)
1. Push your code to GitHub
2. Connect your repository to Vercel
3. Deploy automatically with each push

### Netlify
1. Build the project: `npm run build`
2. Deploy the `out` folder to Netlify

### Other Platforms
The website is a static Next.js application and can be deployed to any static hosting service.
